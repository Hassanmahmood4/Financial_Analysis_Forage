# Financial Analysis Forage

SEC 10-K analysis and a rule-based financial chatbot prototype for Microsoft, Tesla, and Apple (2023-2025).

## Contents

- `financial_10k_data.csv` — extracted 10-K metrics
- `financial_10k_analysis.ipynb` — pandas trend analysis
- `financial_chatbot.py` — command-line chatbot prototype
- `financial_chatbot_submission.zip` — packaged chatbot submission

## Quick start

```bash
python3 financial_chatbot.py --demo
jupyter notebook financial_10k_analysis.ipynb
```
